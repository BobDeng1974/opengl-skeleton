OpenGL Skeleton 
===

A no-fuss skeleton for creating OpenGL applications with SDL2, GLAD, GLM, ASSIMP, STB and Imgui.

1) Clone repo and submodules
```
git clone --recursive --progress -v "https://github.com/zach2good/opengl-skeleton.git"
```
